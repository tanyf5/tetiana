Website Description

This landing is a portfolio website showcasing my skills in creating modern, responsive, and functional websites. My projects are focused on achieving high performance and ease of use for the end users. Here are some key aspects of my site:

  • HTML5 and CSS3 : Core technologies for structuring and styling pages, ensuring modern and clean markup with high accessibility
  • JavaScript (ES6) : Used for interactive elements of the site, such as animations, sliders, and other dynamic features

Responsiveness

  • Mobile Devices: The container's maximum width is 375px
  • Small tablets: The container's maximum width is 744px
  • Large tablets: The container's maximum width is 1024px
  • Desktops: The container's maximum width is 1440px

Interactive Elements

  • Vanilla Tilt: A lightweight library for creating 3D effects when hovering over elements
  • Typed.js: For creating animated text on the site, adding dynamism
  • SweetAlert2: For implementing beautiful and user-friendly pop-up windows (alerts)
  • EmailJS: For integrating contact forms, allowing users to send messages without server-side configuration

Animations and Effects

  • CSS animations are used for smooth color transitions, as well as hover transitions for buttons and other elements
  • A progress bar shows the page load progress, adding interactivity and improving the user experience

Optimization

  • Font and image optimization for fast page loading
  • Adaptive scrollbar width for different browsers
  • Advanced CSS techniques like font-smoothing and text-rendering are applied to ensure high-quality text rendering on all devices

Dark Theme

  • The website supports a dark theme for a more comfortable viewing experience in low light conditions. All styles automatically change using CSS variables, allowing for efficient management of colors and backgrounds

External Libraries

  • For user interaction, SweetAlert2 is used for pop-ups, and EmailJS integrates with the contact form on the site
